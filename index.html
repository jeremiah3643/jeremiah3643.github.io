Project Forum

I built this application to test my abilities. Using ReactJS to build a complete Front-End application, I was testing my ability to replicate a Full-Stack application. I used JSON server as a stand in for a Back-End. I also styled my application using SemanticUI CSS classes. Project Forum is essentially a clean looking forum where users can create an account and post topics and post inside those topics. Users are also given the ability to follow a topic and have posts inside that topic show up in their own homepage.
